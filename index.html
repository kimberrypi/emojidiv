Emoji DIV

A Hacktoberfest Project. (inspired by a.single.div and @hj_chen )

GUIDELINES:

  1. Fork this repository.
  2. Create a div using your name as the ID.
  3. Select an emoji to make.
  4. Write a single div representation of your emoji. You can see a lot of emojis here.
  5. Create a branch, push your work and create a pull request.
  6. Most importantly, enjoy!

Cheat Sheet:

  1. Create a branch:
    git checkout -b {branch-name}
  2. Commit a branch:
    git add .
    git commit -m "add an emoji"
  3. Push a branch to remote:
    git push origin {branch-name}
Copy the div with class emoji and start drawing your chosen emoji with CSS! Don't forget to add your name!
